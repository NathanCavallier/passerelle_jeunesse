# **App Name**: Passerelle Jeunesse

## Core Features:

- Service Overview: Display a summary of available services (transport, education, autonomy support, etc.) tailored for families and youth organizations.
- Secure Contact Form: Implement a contact form for inquiries, ensuring data protection and privacy for sensitive family and youth information.
- Testimonial Display: Showcase testimonials from families and organizations highlighting positive experiences with Passerelle Jeunesse services.
- Scheduling and Coordination: Allow families and organizations to request services through the website. Collect the contact information securely.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) to evoke trust and security.
- Background color: Very light blue (#F0F8FF) for a calm, supportive feel.
- Accent color: Muted teal (#3CB371) for calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif to combine modernity and a touch of warmth.
- Use simple, clear icons to represent each service offering, such as a stylized bus for transportation, or a graduation cap for educational support.
- Maintain a clean, accessible layout that emphasizes ease of navigation and readability across all devices.
- Use subtle animations and transitions to enhance user engagement.

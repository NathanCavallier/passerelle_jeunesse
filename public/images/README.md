# Images

## Logo

Placer ici le logo de Passerelle Jeunesse sous le nom `logo.png`.

Le logo devrait être au format PNG avec fond transparent pour une meilleure intégration.

Dimensions recommandées :

- **Header/Desktop** : 600x600px (sera redimensionné automatiquement)
- **Favicon** : 512x512px
- **Mobile** : 180x180px

## Structure

```
public/
  images/
    logo.png         # Logo principal
    logo-light.png   # Version claire (optionnelle pour dark mode)
    favicon.ico      # Favicon généré depuis le logo
```
